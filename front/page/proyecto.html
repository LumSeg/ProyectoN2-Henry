Sobre el poryecto

Descripción Breve

Propósito: Este proyecto tiene como objetivo crear una aplicación web para gestionar información de películas, permitiendo a los usuarios visualizar, agregar y administrar datos de películas. Tecnologías Utilizadas: Se utilizó Express.js y MongoDB para el backend, junto con Bootstrap 5 y Axios para el frontend, asegurando una experiencia de usuario intuitiva y moderna.


Funcionalidades Principales

Listado de Películas: Se ofrece una vista donde se muestran todas las películas almacenadas en la base de datos MongoDB.
Creación de Películas: Los usuarios pueden agregar nuevas películas mediante un formulario interactivo en el frontend, validando los datos obligatorios antes de enviarlos al backend.
Diseño Responsivo: Se implementó Bootstrap 5 para garantizar que la aplicación sea completamente responsive y accesible desde múltiples dispositivos y tamaños de pantalla.


Estructura del Proyecto

Backend: Organizado en diferentes carpetas como "config", "controllers", "middleware", "models", "router", "services", y "utils", utilizando Express.js para manejar rutas y operaciones CRUD con Mongoose para la interacción con la base de datos."
Resolución de Problemas: Se enfrentaron desafíos como configuraciones de Webpack, manejo de errores en las peticiones HTTP, y validación de datos tanto en el frontend como en el backend, los cuales fueron abordados y resueltos con éxito.


Aprendizajes y Desafíos

Integración Frontend y Backend: Una de las principales lecciones fue aprender a integrar eficazmente el frontend con el backend, asegurando la comunicación correcta entre ambas partes mediante peticiones HTTP y manejo de respuestas.
Frontend: Estructurado en carpetas "page", "public", "script", y "style", con Webpack para el empaquetado de scripts y estilos, Axios para manejar solicitudes HTTP al backend, y jQuery para manipulación del DOM y eventos."


Resultado Final

Éxito y Funcionalidad: El proyecto concluyó con una aplicación completamente funcional que permite a los usuarios administrar información de películas de manera eficiente y amigable, ofreciendo una experiencia fluida desde la interfaz hasta la base de datos.
